Входные данные для функции: ссылка на массив строк каждая строка является числом в десятичной системе счисления  
выходные данные: ссылка на мапу массивов где ключ это строка представляющая сумму цифр числа а значение это массив всех чисел из исходного массива которые при сложении цифр дают эту сумму  
все числа должны быть уникальными в исходном массиве одинаковые числа должны рассматриваться только один раз  
значения в массиве результата должны быть отсортированы в лексикографическом порядке  
ключи мапы должны быть представлены в виде строк сумма цифр равна трём результат должен содержать только те группы чисел где их больше одного  
все числа меньше десяти игнорируются в обработке  
например для чисел пятьдесят один и пятнадцать сумма цифр равна шести и оба числа попадают в одну группу
